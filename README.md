# AI-агент для обработки текстовых обращений

## 📌 Описание

Сервис для автоматической обработки текстовых обращений:
- ✅ Классификация типов запросов (жалоба, вопрос, отзыв, техническая проблема)
- ✅ Суммаризация текста
- ✅ Выделение ключевых пунктов
- ✅ Определение тональности (positive, negative, neutral)
- ✅ Структурированный JSON-вывод
- ✅ Сохранение истории обращений
